## Dependencies:
```
qt6-base qt6-declarative sysinfo(in freebsd)
```

## Build:

```sh
git clone https://github.com/SrWither/guifetch
cd guifetch
mkdir build && cd build
cmake ..
make
./guifetch
```

#### Tested on:

```
ArchLinux FreeBSD and VoidLinux
```
